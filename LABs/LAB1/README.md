## 实验效果评分

对于两个任务，我们将同时考虑多种评估指标：

+ 线性回归
  + $MAE$，即绝对误差，越小越好
  + $R^2$，相关系数，越接近1越好，若为负数，说明你的模型与实际趋势相反
+ 逻辑回归
  + $F1$，越接近1越好
  + $AUC$，$ROC$曲线的面积，越接近1越好
